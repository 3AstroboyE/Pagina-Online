Kuxi, algo sobre la depresión


Lo que busca este programa es tratar y ayudar a los jóvenes tristes y que puedan tener deprecion para que puedan mejorar con mini retos o tareas simples con la intención de tratar de animarlos obviamente con cosas fáciles y que no los afecten negativamente para esto se basara en estudios ya hechos y encuestas.

Se basará de un filtro que catalogara y separara las tareas que se pueden hacer y las que no por ejemplo si no tiene familia no se le podrá poner una tarea relacionada a esta.

Despues del filtro se catalogará con un color para poder llevar a cabo un progreso que el usuario podrá ir cambiando por el mismo usuario para que el programa pueda darle tareas con base a su estado de ánimo con el paso de los días para que pueda tener un control semanal de cómo son sus emociones con el pasar de las semanas y si el programa hizo algo diferente o todo siguió igual.

La idea es que tenga un apartado para poder decir tu problema a una persona anónima y desconocida también la oportunidad de agregar a tus propios amigos y subir los retos que hayas superado para que tus conocidos puedan competir entre sí de manera sana Contando con el calendario para poder cambiar el color del estado de animo.
